# ☿ Alquimia App

App de gestión para Alquimia Productora Gastronómica.

## Cómo subir a Vercel (10 minutos)

### Paso 1 — GitHub
1. Ve a **github.com** y crea una cuenta gratis
2. Haz clic en **New repository**
3. Nombre: `alquimia-app` → clic en **Create repository**
4. Sube todos estos archivos arrastrándolos al repositorio

### Paso 2 — Vercel
1. Ve a **vercel.com** → Sign up with GitHub
2. Clic en **Add New Project**
3. Selecciona el repositorio `alquimia-app`
4. Framework: **Create React App** (lo detecta automático)
5. Clic en **Deploy**
6. En 2 minutos tendrás tu URL: `alquimia-app.vercel.app`

### Paso 3 — Instalar en celular
1. Abre la URL en **Chrome** (Android) o **Safari** (iPhone)
2. Android: menú ⋮ → "Agregar a pantalla de inicio"
3. iPhone: botón compartir → "Añadir a pantalla de inicio"

### Paso 4 — Instalar en computador
1. Abre la URL en **Chrome**
2. En la barra de direcciones aparece un ícono de instalar (monitor con flecha)
3. Clic → "Instalar" → queda en el escritorio

## Datos
Todos los datos se guardan en el navegador (localStorage).
Son privados, solo en tu dispositivo.
